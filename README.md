## About this Project

On The Way fitness tracker was built as a comprehensive workout management system with an intuitive mobile-first design. Users can discover pre-built routines or create their own, log workouts with detailed set/rep tracking, monitor body measurements, and visualize progress through interactive charts and calendars.

## Features

- Create and save custom workout routines
- Browse exercise library with filtering
- Real time work out logger with timer
- Track body measurements over time
- Visual progress statistics and trends
- Calender view of workout history
- Responsive design
